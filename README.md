# SmartTracker

## Installation Kali :
* `apt update`
* `apt install git curl -y`
* `git clone git://github.com/htr-tech/track-ip.git`
* `cd track-ip`

## Run :
* `bash trackip`


## Single Command :
```
apt update ; apt install git curl -y ; git clone git://github.com/htr-tech/track-ip.git ; cd track-ip ; bash trackip
```
